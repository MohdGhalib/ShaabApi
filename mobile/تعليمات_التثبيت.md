# تعليمات تثبيت تطبيق الشعب

## الخطوات

### 1. تثبيت Flutter SDK
- حمّل Flutter من: https://flutter.dev/docs/get-started/install/windows
- فكّ الضغط واضف المسار `flutter\bin` إلى متغير PATH
- شغّل: `flutter doctor` للتأكد من اكتمال الإعداد

### 2. تثبيت Android Studio
- حمّل Android Studio من: https://developer.android.com/studio
- ثبّت Android SDK + Android Emulator أو وصّل هاتف Android حقيقي

### 3. ضبط رابط السيرفر
افتح الملف:
```
shaab_app/lib/constants.dart
```
غيّر السطر:
```dart
const String kBaseUrl = 'https://YOUR-APP.up.railway.app';
```
إلى الرابط الحقيقي لتطبيقك في Railway، مثال:
```dart
const String kBaseUrl = 'https://sh-production-xxxx.up.railway.app';
```

### 4. بناء التطبيق
افتح Terminal في مجلد `shaab_app` وشغّل:
```bash
flutter pub get
flutter build apk --release
```

ملف APK سيكون في:
```
shaab_app/build/app/outputs/flutter-apk/app-release.apk
```

### 5. تثبيت APK على الهاتف
انقل الملف للهاتف وثبّته (يجب تفعيل "مصادر غير معروفة" في الإعدادات)
